About Small Tales


Small Tales is a little web app created by Adam Barak. The source code can be viewed here.

The purpose of this site is to provide a place to read and collaboratively create tales with other users.

The basic idea is that someone starts a tale, and then a variety of people add phrases to it (generally consisting of a sentence or two) to create fun and often strange tales for all to read.
